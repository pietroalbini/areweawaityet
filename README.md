# Are we await yet?

This website tracks the bickeshedding of [Rust](https://www.rust-lang.org)'s
[async/await](https://areweasyncyet.rs) feature, and especially the syntax of
the `await` keyword. The repository is released under the MIT license.

## Running

To generate the website you need to run:

```
$ cargo run
```

The generated content will be present in the `static` directory.
